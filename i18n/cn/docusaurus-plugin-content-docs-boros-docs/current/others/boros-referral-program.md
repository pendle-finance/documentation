# Boros 推荐计划

<figure><img src="/boros-docs/imgs/image.png" alt="" /><figcaption></figcaption></figure>

您可以在 Boros 的账户页面访问推荐页面：[https://boros.pendle.finance/account](https://boros.pendle.finance/account)

* 使用推荐码将为您提供 10% 的交换费折扣。
* 推荐人将获得该推荐码产生的所有费用的 20%。

## 资格要求

我们鼓励新地址使用推荐码！交易量超过 100,000 美元名义价值的地址将不再有资格使用推荐码。

要生成推荐码，您的地址必须至少拥有 **1,000,000 美元的名义交易量**。

Boros 推荐系统的上限：

* 每个地址限于 1,000,000,000 美元的名义交易量，超过后 10% 折扣将不再适用。
* 每个推荐码从生成之日起有效期为 1 年。

<Hint style="warning">
Boros 团队保留确定推荐码资格和支付的权利。被团队认定为疑似滥用的情况（例如专门自我推荐）将导致无法支付或取消推荐码。
</Hint>
