# Pendle 101 - 核心要点

> 课程级别：**#初级**

## 1. **Pendle 优化收益入门**

* 收益代币是从各种 DeFi 协议中产生收益的代币。
* Pendle 是一个使用户获得更好收益的平台，通过以下方式实现：
  * 收益代币化
  * 收益交易
* 以下是您可以在 Pendle 上获得更好收益的几个示例：
  1. 赚取固定收益
  2. 无需额外风险，使用您已有的相同资产赚取额外收益（流动性提供）
  3. 通过购买更多收益来增加收益敞口
  4. 以及更多～

## 2. **收益代币化基础**

### 收益代币化

* 这意味着将收益资产拆分为两部分：**本金**和**收益**。
* **本金** + **收益** = **收益资产**
* Pendle 是一个市场，收益资产可以在这里被代币化为独立的本金和收益代币，并分别进行交易。

### PT 与 YT

Pendle 将收益代币拆分为两部分：**本金代币（PT）** 和 **收益代币（YT）**。

#### **本金代币（PT）**

* **PT 让您在到期日后赎回底层本金资产。**
* PT 的**入场成本低于**原始资产。PT 的价值随时间增长，到期时可 1:1 赎回原始资产。
* 您可以通过[买入并持有 PT 来赚取固定收益](../optimizing-yields-with-pendle/chapter-3.1-fixed-yield-on-pendle)。入场成本与赎回价值之间的差额即为您的固定收益。
* PT 类似于传统金融中的[零息债券](https://www.investopedia.com/terms/z/zero-couponbond.asp)。

#### **收益代币（YT）**

* **YT 让您在到期日前获得底层资产的收益，可实时领取。**
* 您可以以远低于本金的价格购买收益或更多收益。如果您获得的收益超过购买 YT 的成本，则获利。
* 通过购买 YT，您可以获得[杠杆收益敞口](../yield-trading-deep-dives/chapter-8-long-yield-obtain-leveraged-yield-exposure)，无需承担清算或预言机风险。
* YT 类似于传统金融中的[债券分离息票](https://www.investopedia.com/terms/c/coupon.asp)。

<Hint style="danger">
**您可以随时出售 PT 和 YT**，在 Pendle 市场上，_**无需**_**锁定或罚款**，以市价随时交易，24/7 全天候可用。
</Hint>
