# 领取

## 领取 YT 累积的收益以及挖矿奖励

1. 前往 [Dashboard](https://app.pendle.finance/pro/dashboard/overview)
2. 点击「**领取收益和奖励**」("Claim Yield and Rewards")
3. 使用右上角的滑块按余额金额筛选
4. 选择要领取的奖励
5. 使用左上角的切换按钮在 USDC 和原生代币之间选择
6. 确认 gas 费和兑换手续费后，点击领取！
