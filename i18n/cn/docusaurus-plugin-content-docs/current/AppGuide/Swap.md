# 兑换

## 通过 PT 获取固定收益

1. 前往 [Markets](https://app.pendle.finance/pro/markets/)。

![PT Swaps 1](/pendle-docs/imgs/AppGuide/pt-swaps-1.png)

2. 点击 PT 栏，选择目标资产。

![PT Swaps 2](/pendle-docs/imgs/AppGuide/pt-swaps-2.png)

3. 确认处于「**兑换**」("Swap") 标签页，然后选择输入资产。

![PT Swaps 3](/pendle-docs/imgs/AppGuide/pt-swaps-3.png)

4. 输入交易金额。可通过右上角的齿轮图标调整滑点容忍度。

![PT Swaps 4](/pendle-docs/imgs/AppGuide/pt-swaps-4.png)

5. 查看输出信息：
  * 最大滑点：可容忍的最大滑点。超过此容忍值时交易将回滚。
  * 实际固定 APY：考虑价格影响和兑换手续费后，此次交易实际获得的固定 APY，等同于 PT 的最终执行价格/平均买入价格。

![PT Swaps 5](/pendle-docs/imgs/AppGuide/pt-swaps-5.png)

6. 准备就绪后，批准交易并兑换！

## 通过 YT 扩大收益敞口

1. 前往 [Markets](https://app.pendle.finance/pro/markets/)。

![YT Swaps 1](/pendle-docs/imgs/AppGuide/yt-swaps-1.png)

2. 点击 YT 栏，选择目标资产。

![YT Swaps 2](/pendle-docs/imgs/AppGuide/yt-swaps-2.png)

3. 确认处于「**兑换**」("Swap") 标签页，然后选择输入资产。

![YT Swaps 3](/pendle-docs/imgs/AppGuide/yt-swaps-3.png)

4. 输入交易金额。可通过右上角的齿轮图标调整滑点容忍度。

![YT Swaps 4](/pendle-docs/imgs/AppGuide/yt-swaps-4.png)

5. 查看输出信息：
  * 最大滑点：可容忍的最大滑点。超过此容忍值时交易将回滚。
  * 实际固定 APY：考虑价格影响和兑换手续费后，此次交易实际获得的固定 APY，等同于 YT 的最终执行价格/平均买入价格。
  * 名义交易价值：产生此次交易所需的底层资产数量，也可理解为收益敞口规模（即 YT 产生于等值 $122,366 DAI 存款的收益）。

![YT Swaps 5](/pendle-docs/imgs/AppGuide/yt-swaps-5.png)

6. 准备就绪后，批准交易并兑换！
