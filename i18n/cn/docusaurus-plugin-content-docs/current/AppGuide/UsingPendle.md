# 使用 Pendle

Pendle App 为用户提供 Pendle 全套功能，可以自由交易收益并执行任意收益策略。

用户可以将生息资产存入 Pendle，铸造本金代币（**PT**）和收益代币（**YT**）。PT 代表底层生息代币的本金，YT 代表对资产**全部**收益的权利。PT 和 YT 均可在 Pendle 上交易。

![Yield splitting](/pendle-docs/imgs/AppGuide/yield_splitting.png "Yield splitting")

在传统金融中，Pendle 的做法类似于债券剥离 (bond stripping)——本金与利息被分离。因此，PT 等同于[零息债券](https://www.investopedia.com/terms/z/zero-couponbond.asp)，而 YT 则等同于被剥离的[票息](https://www.investopedia.com/terms/c/coupon.asp)。

## 策略

### 买入 PT

若你认为资产收益率将下降，可通过买入 PT 来**对冲收益**。由于到期后保证获得底层资产，买入 PT 实际上是以当前 Implied APY 锁定了你的年化收益率，即**固定你的收益**。

例如，以 5% 隐含收益率买入 1 年期 PT-aUSDC，意味着每花费 1 个 USDC 购买的 PT，到期赎回后将获得 1.05 个 USDC。

### 买入 YT

另一方面，若你认为资产收益率将上涨，可通过买入 YT 来**押注收益**。买入 YT 意味着以仅购买收益部分的方式扩大对某资产收益的敞口，回报由底层 APY 的波动决定。

此外，买入 YT 比直接买入底层资产资本效率更高——相同的资金，可以购买多得多的 YT 数量，从而**成倍放大收益敞口**。

例如，若 YT 价格为底层资产价格的 5%，底层收益率的任何上涨都将带来 20 倍的回报放大，因为你可以购买 20 倍数量的 YT。

### 提供流动性

若你认为某资产的收益率不太可能大幅波动，可以直接向我们的池子提供流动性，通过兑换手续费和激励获得额外收益。

与大多数其他收益协议不同，我们的池子不存在由底层资产价格波动引起的无常损失风险，因为 PT 和 YT 的价格与底层资产价格挂钩（PT + YT = 底层资产）。唯一的无常损失风险来自 PT 和 YT 需求的波动，这是所有流动性池固有的风险。

若流动性持续提供至到期，无常损失有上限且趋向最小。这是可行的，因为两种资产在到期时价值完全相同。因此，提供流动性也可以作为任何 PT 或 YT 仓位的对冲工具。
